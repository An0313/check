# @ahg/check 常用校验

##### 身份证号校验

支持15位与18位

```javascript
import {ac_idCardCode} from '@ahg0313/check'

ac_idCardCode('身份证号码')
```


#### 客户端是否是安卓
```javascript
import {ac_isAndroid} from '@ahg0313/check'

ac_isAndroid()
```


#### 客户端是否是ios
```javascript
import {ac_isIOS} from '@ahg0313/check'

ac_isIOS()
```


#### 手机号校验
```javascript
import {ac_isMobileCode} from '@ahg0313/check'

ac_isMobileCode('手机号')
```
